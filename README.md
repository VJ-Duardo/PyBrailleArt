# braille-unicode-inverter

This script invertes unicode-art made up of braille characters to the opposite "color".

For example this:

<p>⣿⣿⣿⣿⣿⣿⣿⡿⠟⠛⠉⠉⠉⠉⠋⠉⠉⠙⠛⠛⠻⠿⢿⣿⣿⣿⣿⣿⣿⣿ <br> 
⣿⣿⣿⣿⠿⠋⠁⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠈⠉⠻⣿⣿⣿⣿⣿ <br> 
⣿⣿⡟⠁⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠙⢻⣿⣿⣿ <br> 
⣿⠏⠄⠄⠄⠄⠄⠄⠄⠄⢀⣔⢤⣄⡀⠄⡄⡀⠄⠄⠄⠄⠄⠄⠄⠄⠄⢻⣿⣿ <br> 
⠏⠄⠄⠄⠄⠄⠄⠄⢀⣀⣨⣵⣿⣿⣿⣿⣧⣦⣤⣀⣿⣷⡐⠄⠄⠄⠄⠄⢿⣿ <br> 
⠄⠄⠄⠄⠄⠄⠐⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⣦⡀⠄⠄⠄⢚⣿ <br> 
⣆⠄⠄⠄⠄⠄⠄⢻⣿⣿⣿⣿⡿⠛⠛⠛⠛⣿⢿⣿⣿⣿⡿⢟⣻⣄⣤⣮⡝⣿ <br> 
⣿⠆⠄⠄⠄⠄⠄⠄⠄⠄⠉⠘⣿⡗⡕⣋⢉⣩⣽⣬⣭⣶⣿⣿⣿⣿⣝⣻⣷⣿ <br> 
⣿⣦⡀⠄⠄⠠⢀⠄⠄⠁⠄⠄⣿⣿⣶⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⡟⣿ <br> 
⣿⣿⣿⡆⠄⠄⠰⣶⡗⠄⠄⠄⣿⣿⣿⣿⣦⣌⠙⠿⣿⣿⣿⣿⣿⣿⣿⡛⠱⢿ <br> 
⣿⣿⣿⣿⡀⠄⠄⠿⣿⠄⠄⠄⠨⡿⠿⠿⣿⣟⣿⣯⣹⣿⣿⣿⣿⣿⣿⣿⣦⡀ <br> 
⣿⣿⣿⣿⣷⠄⠄⠄⢷⣦⠄⠄⠐⢶⢾⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⡇ <br> 
⣿⣿⣿⣿⣿⣧⡄⠄⠄⠉⠄⠄⠄⢉⣽⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⣿⠟⠄ <br> 
⣿⣿⣿⣿⣿⠟⠋⠄⠄⠄⠄⠄⠄⠈⠛⠿⠿⣿⣿⣿⣿⣿⣿⣿⣿⣿⡿⠋⠄⠄ <br> 
⣿⠿⠛⠉⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠘⠿⢿⣿⣿⣿⣿⣿⠿⠋⠄⠄⠄⠄</p> 

will be converted to this:

<p>⠄⠄⠄⠄⠄⠄⠄⢀⣠⣤⣶⣶⣶⣶⣴⣶⣶⣦⣤⣤⣄⣀⡀⠄⠄⠄⠄⠄⠄⠄ <br> 
⠄⠄⠄⠄⣀⣴⣾⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣷⣶⣄⠄⠄⠄⠄⠄ <br> 
⠄⠄⢠⣾⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣦⡄⠄⠄⠄ <br> 
⠄⣰⣻⣻⣻⣻⣻⣻⣻⣻⡿⠫⡛⠻⢿⣻⢻⢿⣻⣻⣻⣻⣻⣻⣻⣻⣻⡄⠄⠄ <br>  
⣰⣻⣻⣻⣻⣻⣻⣻⡿⠿⠗⠊⠄⠄⠄⠄⠘⠙⠛⠿⠄⠈⢯⣻⣻⣻⣻⣻⡀⠄ <br> 
⣻⣻⣻⣻⣻⣻⣯⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠙⢿⣻⣻⣻⡥⠄ <br> 
⠹⣻⣻⣻⣻⣻⣻⡄⠄⠄⠄⠄⢀⣤⣤⣤⣤⠄⡀⠄⠄⠄⢀⡠⠄⠻⠛⠑⢢⠄ <br> 
⠄⣹⣻⣻⣻⣻⣻⣻⣻⣻⣶⣧⠄⢨⢪⠴⡶⠖⠂⠓⠒⠉⠄⠄⠄⠄⠢⠄⠈⠄ <br> 
⠄⠙⢿⣻⣻⣟⡿⣻⣻⣾⣻⣻⠄⠄⠉⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⢠⠄ <br> 
⠄⠄⠄⢹⣻⣻⣏⠉⢨⣻⣻⣻⠄⠄⠄⠄⠙⠳⣦⣀⠄⠄⠄⠄⠄⠄⠄⢤⣎⡀ <br> 
⠄⠄⠄⠄⢿⣻⣻⣀⠄⣻⣻⣻⣗⢀⣀⣀⠄⠠⠄⠐⠆⠄⠄⠄⠄⠄⠄⠄⠙⢿ <br> 
⠄⠄⠄⠄⠈⣻⣻⣻⡈⠙⣻⣻⣯⡉⡁⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⢸ <br> 
⠄⠄⠄⠄⠄⠘⢻⣻⣻⣶⣻⣻⣻⡶⠂⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⣠⣻ <br> 
⠄⠄⠄⠄⠄⣠⣴⣻⣻⣻⣻⣻⣻⣷⣤⣀⣀⠄⠄⠄⠄⠄⠄⠄⠄⠄⢀⣴⣻⣻ <br> 
⠄⣀⣤⣶⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣻⣧⣀⡀⠄⠄⠄⠄⠄⣀⣴⣻⣻⣻⣻</p> 

## Usage

Just paste the string into the `input_str` variable and run the script. It should now print out the inverted string.
